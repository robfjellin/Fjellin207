# First Person Shooter
A fast paced arena shooter developed as part of the GAM207 course at Southern New Hampshire University.

## About
This project is an arena shooter where the player faces off against enemies that spawn at one of three spawn locations. The game focuses on core shooting mechanics, aiming, shooting, and enemy AI behavior.

## Features
* Player movement and controls
* Shooting mechaniccs with projectile/hit detection
* Enemy AI that detects players within LOS
* Scoring system
* Arena environment

## Controls
* WASD - Movement
* Mouse - Aim
* Left Click - Fire
* Space - Jump
* Tab - Scoreboard

## Technologies Used
* Engine: UE5
* Language: C++, Blueprints

## Development Notes
This project was created to gain a better understanding of:
* Game mechanics implementation
* AI Programming
* Level Design
* Player feedback
